for idx, feat in enumerate(legit.columns):
    q75, q25 = np.percentile(legit[feat], [75 ,25])

Implementation¶

Here, I'll be running an anomaly detection approach.
I'll be using the Gaussian Mixture Model provided by scikit-learn and the 't Mixture Model' implemented by me, but proposed by D. Peel and G. J. McLachlan (Robust Mixture modelling using the t distribution).
Most of the implementations were based on the scikit-learn GaussianMixture.
